Role Name
=========

Role to install the server side components of a Simple Logging solution for CentOS/RHEL 7 based on Elasticsearch, Logstash,and Kibana (Elastic Stack).

Requirements
------------

A system with the following specs:
- CentOS/RHEL 7
- Online connectivity
- At least 8 GB of RAM 
- At least 2 CPU
